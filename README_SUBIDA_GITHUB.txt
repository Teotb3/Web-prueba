VERSIÓN SIMPLE PARA GITHUB PAGES

Todos los archivos están en la misma carpeta.

Para subirlo a GitHub Pages:
1. Entra en tu repositorio.
2. Borra las carpetas antiguas assets, icons, images, logos, videos si existen.
3. Sube TODOS los archivos de esta carpeta directamente en la raíz del repositorio.
4. Asegúrate de que index.html queda en la raíz, no dentro de otra carpeta.
5. Haz commit.

Importante: no cambies los nombres de los archivos después de subirlos, porque las rutas del HTML ya están actualizadas para estos nombres.
