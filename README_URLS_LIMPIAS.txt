Notas importantes:
1. En GitHub Pages o en teotorres.com las URLs limpias funcionan por la estructura de carpetas.
2. Para probar en local abriendo el archivo con doble clic, los enlaces están preparados para abrir los index.html internos y no romperse.
3. Las rutas de imágenes, vídeos y favicons se han dejado relativas para que funcionen tanto en local como online.
4. Los archivos .html antiguos siguen existiendo como redirecciones para no romper enlaces previos.
